# vigilo
My personal Cybersecurity and Networking tools library. 

**Disclaimer: These tools are for education and testing purposes only.**
They are intended to be used in controlled, sandbox environments to help learn how network protocols and security vulnerabilities work.

**Do not use these tools on any system or network you do not own or have explicit permission to test. Using these tools on a live, third-party network can be illegal. The author is not responsible for any misuse or damage caused by this code.**
